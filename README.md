# agenda_contatos

Este foi feito em Flutter.

## Sobre

Ele suas funcionalidade basicas de uma agenda de contatos.

Exemplo:

- Cria contato
- Deleta contato
- Atualizar contato
- Adiciona fotos do contato
- Ordena contato
